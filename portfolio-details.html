Matrix-Based Image Processing

This project implements fundamental image processing operations using matrix-level operations with NumPy and OpenCV. The approach focuses on custom convolution kernels, pixel-wise transforms, and efficient array manipulations to create sharpen, blur, edge-detection and rotation functions without relying purely on high-level wrappers.

The codebase includes unit-tested filter functions, example notebooks demonstrating step-by-step operations, and a small Flask demo that applies selected filters to user-uploaded images. This project helped deepen my understanding of convolution, kernel design and performance optimisation using vectorised NumPy operations.

“The matrix-first approach made filter creation explicit and faster to prototype. Great learning experience about low-level image operations.”

The repository includes sample images, a README with usage instructions, and a small demo app to test filters in the browser. The next steps are to containerize the demo and add GPU-accelerated operations for larger images.
